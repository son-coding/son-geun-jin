전체화면

Description

브랜드 쇼핑몰 홈페이지를 소개합니다.

크게 3부분으로 이루어지며,소비자의 구매심리 자극과

이용 편의성에 초점을 두고 만들었습니다.

code review:

html code를 만들 때 body태그 안에 크게

header (상단 메뉴)

section (중간 컨텐츠)

footer (하단 메뉴)

이렇게 세가지 부분으로 나누어 구성했습니다.

상단 메뉴

Description

상단 메뉴에는 중앙 로고를 기준으로 왼쪽은 클릭으로 열리는 사이드 메뉴가 있고 오른쪽은 로그인, 회원가입, 검색 칸이 있습니다. 이 상단 메뉴는 스크롤을 해도 위에 고정되어 있어서 로고가 항상 보이도록 하였고, 스크롤 하여 상품을 보다가 언제든지 로그인을 하거나 다른 상품을 검색할 수 있게 하였습니다.

code review:

header태그 안에 개체들을 4개의 그룹으로 나누어 로고에 왼쪽 마진값을 화면에 반정도 차지하게 만들어 중앙에 옮기고 이를 기준으로 나머지 개체들을 각각 간격을 주어 위치 시켰습니다.

중간 컨텐츠

Description

중간 컨텐츠는 4부분으로 나누어집니다.

1. 메인 이미지에 유명연예인의 상품착용샷
2. 동기부여 영상과 행사 상품 슬라이드 쇼
3. 묶음상품(할인된 가격에 구매가능)
4. 유행 상품 게시하여 소비자가 유행에 맞춰 상품을 구매 가능

code review:

section 태그 안에 container 그룹을 4개 만들어 그룹 간에 간격을 주고 구분을 하였습니다. 나중에 원하는 container에서 물건을 변경할 수 있고 새로운 container를 추가하거나 삭제하기 편하게 만들었습니다.

하단 메뉴

Description

마지막 하단 메뉴에는 신상품과 다른 종류의 물건을 구분을 해서 클릭시 해당 페이지로 이동할 수 있게 하였습니다. Top버튼은 클릭시 맨위 화면으로 자동 스크롤 해주는 버튼으로 스크롤을 조금만 내리면 오른쪽 하단에 표시되고 고정되게 만들었습니다.

code review:

header code와는 다르게 같은 성격에 항목을 목록화하려고 li태그를 사용하였고 횡방향으로 정렬하여 각각 마진값을 주어 간격을 유지하였습니다.
